## What this is
A calm, ADHD-friendly, local-first style **Founder Operating System portal** built with:
- Plain HTML/CSS/JS (no framework)
- Deterministic structure
- GitHub-first workflow (manual, one file at a time)
- Designed as a **showcase** for our workflow + UX
